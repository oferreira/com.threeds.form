Installation

Include required files : Call jquery and header3ds plugin (js and css):

If you already use jQuery, you can use your version 

<script src="http://www.3ds.com/header3ds/jquery-1.10.2.min.js"></script>
<script src="http://www.3ds.com/header3ds/jquery.header3ds.js"></script>
<link rel="stylesheet" href="www.3ds.com/header3ds/jquery.header3ds.css" />

or minified versions - recommanded! 

<script src="http://www.3ds.com/header3ds/jquery-1.10.2.min.js"></script>
<script src="http://www.3ds.com/header3ds/jquery.header3ds.min.js"></script>
<link rel="stylesheet" href="http://www.3ds.com/header3ds/jquery.header3ds.min.css" />

Setup

You can call header without option (defaults values)

$('body').header3ds();

customization

You can also customize layout :

$('body').header3ds({
	"bgcolor" 		: "light",
	"website"		: "g",
	"hasfooter"		: true,
	"hasborder"		: true,
	"hascompass"		: true,
	"language"		: "fr",
	"secure"		: true,
	"width"			: "100%",
	"margin"		: "35",
	"mediaqueries"	: [
		{
			"min"		: "1100px",
			"max"		: "1299px",
			"width" 	: "1000px",
			"margin"	: "55px"
		},
		{
			"min"		: "1300px",
			"width" 	: "1200px"
		}
	]
});

bgcolor :

Define the background dominant color

  • possible values :
    • black : #000000
    • dark : #33424b
    • blue : #305164
    • grey : #d1cfcf
    • light : #eaeaea
    • white : #FFFFFF
  • default value : blue

hasborder :

Add / Remove the border

  • possible values :
    • true
    • false
  • default value : true

hascompass :

Add / Remove the Compass

  • possible values :
    • true
    • false
  • default value : true

website

Display the quick access according to your website

  • possible values (this value will be provided by the webteam):
    • a : for website with no quick access
    • b : for website about industries
    • c : for website about industries and products
    • d : for website "passion for innovation"
    • e : for website about partners
    • g : for website about Events
    • ... : TBD
  • default value : a

secure

You can choose to use secure http (https) or not, for all links !

  • possible values :
    • true
    • false
  • default value : false

hasfooter

Define if you want add the 3DS footer or not !

  • possible values :
    • true
    • false
  • default value : false

language

Define language to use

  • possible values (use ISO 639-1):
    • en : english
    • fr : french
    • de : german
    • it : italian
    • es : spanish
    • ja : japanese
    • zh : chinese
    • ko : korean
    • ru : russian
    • sv : swedish
    • pt : portuguese
  • default value : en

width

Define width (from logo to right icons)

  • possible values :
    • XXXpx : for XXX pixels width
      • Caution : not responsivness with "px" values
    • XX% : for XX percent width
      • max width is 100% (a bigger value will be ignored)
  • default value : 100%

callback

To call a "callback function" after than header/footer was added in the DOM

mediaqueries

Define as many custom media queries as needed

  • The mediaqueries parameter is an array, in which each media query is an object, with those four attributes:
    • min: The lower breakpoint of the media query ( with unit )
    • max: The upper breakpoint of the media query ( with unit )
    • width: The desired width of the header ( with unit ) (cf. WIDTH attribute)
    • margin: The horizontal margin for the header, in case of a % width ( with unit ) (cf. MARGIN attribute)
  • If one of the attributes is missing, set to null, or to empty string, it is ignored
  • If a media query range is not defined, the default WIDTH and MARGIN are applied

margin

If you choose a percent (%) width, you can force the horizontal margin (at left of logo and right of icons/compass)

  • possible values :
    • XXX : for XXX pixels margin
  • default value : 0
